USAGE MODES
-----------

1. Collect & Analyze (One-Step):
   python cat.py --collect --os windows --analyze
   python cat.py --collect --os linux --analyze

2. Collect Only:
   python cat.py --collect --os windows --package
   python cat.py --collect --os linux --collection-output ./artifacts

3. Analyze Existing:
   python cat.py --windows-artifacts ./win --linux-artifacts ./lin
   python cat.py --collected-dir ./artifacts/hostname_timestamp

4. Single File:
   python cat.py --single-file file.log --artifact-type sshlogin
